# lambda-api-ct

## Prerequisites

- Installation of terraform.
- Aws accout with credentials/roles

## Procedure

### Step 1 : Clone git Repository

```bash
git clone <repo-url>


### Step 2 : Run terraform init
```bash
terraform init

### Step 2 : Run terraform plan
```bash
terraform plan

### Step 2 : Finally, run terraform apply
```bash
terraform apply

- IAM Roles
- IAM POLICY
- Lambda function
